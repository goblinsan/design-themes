# design-themes

Shared theme guidance, profile catalogs, and token defaults for Kulrs and Vizial.

## Scope

This package centralizes the non-color parts of a theme system:

- Typography pairings
- Spacing scales
- Radius scales
- Depth styles
- Button styles
- Card styles
- Navigation styles
- Contrast profiles
- Intensity profiles
- Theme guidance presets that compose the profiles above

Kulrs can use the guidance presets as palette-aware design directions. Vizial can
use the same presets and catalogs in its style-flow and theme generation flows.

## Local Development

```bash
npm install
npm run build
```
